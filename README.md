# twitch-ranos

## Usage

Command prefix: `!`
Should be run with the following environment variables:
* `TWITCH_USERNAME`: The username of the bot
* `TWITCH_CHANNELS`: Comma separated list of channels this bot should listen to
* `TWITCH_TOKEN`: The token to authenticate the bot with Twitch's servers. Generate with [this link][0].

### Commands

* `hello`: replies with a welcoming response.
* `uptime`: responds with the number of seconds the bot has been up.
* `quit`: shuts down the bot.

[0]: https://twitchapps.com/tmi/
